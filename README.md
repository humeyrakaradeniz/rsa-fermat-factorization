# rsa-fermat-factorization

## Project Overview

This project investigates the effect of the distance between two prime numbers on the security of the RSA cryptosystem.

The main objective is to experimentally analyze whether choosing two close prime numbers makes RSA factorization easier when using Fermat's Factorization Method.

## Objectives

- Implement RSA from scratch using Python.
- Generate prime numbers without relying on built-in RSA libraries.
- Implement Fermat's Factorization Algorithm.
- Measure factorization time for different prime distances.
- Visualize the results using graphs.

## Technologies

- Python
- Jupyter Notebook
- Matplotlib
